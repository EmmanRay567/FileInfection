## Overview

FileInfection is an experimental audio plugin designed to explore real-time audio processing, parameter control, and custom UI behavior using the JUCE framework. The project combines technical DSP concepts with a creative, glitch-oriented visual and sonic style.

---

## Features

- Glitch-style audio processing effects
- Real-time parameter control
- Custom plugin UI
- Standalone and VST3 build targets
- Minimal, performance-focused design
